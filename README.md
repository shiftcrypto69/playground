Salah satu proyek utama dalam repositori ini adalah **Playground Editor**, sebuah lingkungan pengembangan ringan berbasis web untuk pengujian kode secara real-time.

### **Fitur Utama & Fungsi Editor:**

* **Live Preview:** Lihat perubahan kode Anda secara instan tanpa perlu refresh manual.
* **Multi-Language Support:** Mendukung sintaksis untuk HTML, CSS, JavaScript, dan Markdown.
* **Auto-Save Local:** Progress kerja Anda tersimpan secara lokal di browser agar tidak hilang saat tidak sengaja tertutup.
* **Theme Customization:** Mode Gelap (Dark Mode) dan Terang yang nyaman di mata untuk sesi coding lama.
* **Code Snippets Library:** Simpan dan panggil kembali potongan kode favorit Anda dengan cepat.
* **Export to File:** Unduh hasil eksperimen langsung menjadi file `.html` atau `.js`.
* **Console Integrated:** Debugging langsung di dalam editor tanpa harus membuka Developer Tools browser.
